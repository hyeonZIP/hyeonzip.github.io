# 1. Python type()⭐ [24.10]

1. **int**: 정수형 (`type(5)` → `<class 'int'>`)
2. **str**: 문자열 (`type("hello")` → `<class 'str'>`)
3. **float**: 실수형 (`type(3.14)` → `<class 'float'>`)
4. **list**: 리스트 (`type([1, 2, 3])` → `<class 'list'>`)
5. **tuple**: 튜플 (`type((1, 2, 3))` → `<class 'tuple'>`)
6. **dict**: 딕셔너리 (`type({'a': 1, 'b': 2})` → `<class 'dict'>`)
7. **set**: 집합 (`type({1, 2, 3})` → `<class 'set'>`)
8. **bool**: 불리언 (`type(True)` → `<class 'bool'>`)
9. **NoneType**: `None` 객체 (`type(None)` → `<class 'NoneType'>`)
10. **function**: 함수 (`type(lambda x: x + 1)` → `<class 'function'>`)
11. **module**: 모듈 (`type(os)` → `<class 'module'>`, `import os`가 필요)
